Atmosphere Atlas logo asset pack

Recommended files to copy into your Vite public/ folder:
- app-icon.png
- favicon.png
- favicon-32.png
- favicon-16.png
- apple-touch-icon.png
- android-chrome-192x192.png
- android-chrome-512x512.png
- site.webmanifest

Use app-icon.png for the header logo.
Use favicon.png for the browser tab favicon.

Note: logo-embedded.svg is an SVG wrapper that embeds the PNG. It is not a true vector redraw.
For a true editable vector logo, trace/clean the mark manually in Figma, Illustrator, or Inkscape.
